https://gothic-shop.vercel.app/

Gothic Shop is a frontend e-commerce project built with modern React.  
It implements basic e-commerce functionality: catalog browsing, product pages, shopping cart, server-side product filtering and search, checkout, user authentication.

Technologies:
- React + TypeScript  
- Zustand
- React Router
- TanStack Query
- Axios
- FSD
- Vite
- SCSS Modules
- Custom Skeleton Loader (like react-loading-skeleton lib)
- Lucide icons
- js-cookie
- LocalStorage (cart sync between tabs)  
- React Hook Form
- Custom backend (Node.js + Express, REST API)
